# CommitEase

CommitEase is a multi-tenant SaaS platform designed to digitize contribution collection and financial management for masjid committees.

## Core Problem

Many masjids still manage member contributions manually using notebooks, spreadsheets, and cash-based workflows. This creates problems in transparency, due tracking, reporting, and scalability.

CommitEase solves this by providing:

* Digital member management
* Automated due calculation
* Online payment integration
* Cash payment recording
* Financial reporting
* Audit logging

---

## Core Features

### Super Admin Portal

* Create and manage masjids
* Create committee accounts
* Platform-level management

### Committee Portal

* Manage members
* Record cash payments
* View reports
* Track outstanding dues

### Member Portal

* Login via OTP
* View dues
* Pay online
* Download receipts
